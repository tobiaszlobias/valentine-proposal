# valentineproposal

Jednoduchá statická stránka s otázkou „Budeš můj Valentýn?“ – tlačítko **Ne** je schválně „plaché“.

## Spuštění lokálně

- Otevři `index.html` v prohlížeči (stačí dvojklik).

## Sdílení jako odkaz (doporučení)

Nejjednodušší je to dát na **GitHub Pages** (nebo Netlify / Vercel) a pak poslat odkaz.

### GitHub Pages (rychlé)

1. Nahraj repozitář na GitHub.
2. Settings → Pages → „Deploy from a branch“
3. Branch: `main`, folder: `/ (root)`
4. Po chvíli dostaneš veřejný odkaz.
